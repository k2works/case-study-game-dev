■ 起動方法

  【推奨】起動スクリプトをダブルクリックしてください:

  Windows の場合:
    → puyo-puyo-tdd.bat をダブルクリック

  Linux/macOS の場合:
    → puyo-puyo-tdd.sh をダブルクリック
    （ターミナルから: ./puyo-puyo-tdd.sh）

  【代替】コマンドラインから実行:
    java -jar puyo-puyo-tdd-windows-x64-1.0.0.jar


■ 必要環境

  - Java Runtime Environment (JRE) 21 以上
  - ダウンロード: https://adoptium.net/


■ 操作方法

  ← / → キー    : ぷよを左右に移動
  ↑ キー / Z    : ぷよを右回転
  X キー        : ぷよを左回転
  ↓ キー        : ぷよを高速落下
  リセットボタン : ゲームをリセット


■ ルール

  - 4つ以上つながったぷよを消してスコアを獲得
  - 連鎖すると高得点（最大 8連鎖以上で ×160 倍）
  - 全消しで +3600 点のボーナス
  - 新しいぷよが配置できなくなるとゲームオーバー


■ トラブルシューティング

  Q: JAR ファイルをダブルクリックしても起動しない
  A: 起動スクリプト（.bat または .sh）を使用してください

  Q: Java がインストールされているか確認したい
  A: コマンドラインで「java -version」を実行してください

  Q: エラーメッセージが表示される
  A: Java 21 以上がインストールされているか確認してください
