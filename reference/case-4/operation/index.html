    環境別設定¶

    Local Development¶

    • 起動コマンド: npm run dev
    • ポート: 3000
    • Hot Reload: 有効
    • テスト: npm run test

    CI Environment (GitHub Actions)¶

    • Node.js バージョン: 20.x, 22.x (マトリックステスト)
    • 実行OS: Ubuntu Latest
    • キャッシュ: npm cache有効
    • 並列実行: 複数Node.jsバージョン同時実行

    Production (Vercel)¶

    • デプロイトリガー: develop branchへのpush
    • ビルドコマンド: npm run build
    • 出力ディレクトリ: dist
    • CDN: グローバル配信

    監視・ログ¶

    テストカバレッジ監視¶

    • サービス: Codecov
    • レポート: プルリクエスト時自動コメント
    • しきい値: 80%目標（現在80.57%達成済み）

    品質メトリクス¶

    • サイクロマティック複雑度: 最大7
    • 型安全性: TypeScript strict mode
    • アーキテクチャ違反: dependency-cruiser自動検出
